MySQL5.5 ---2003错误

发表于 2019-07-11 | 分类于 MySQL | | 阅读次数: | 浏览 次

错误信息


解决方案

找到MySQL的安装目录
找到bin目录下的mysqld.exe

双击运行，在MySQL5.5 Command Line Client,输入账号密码就可以使用了

分析问题

出现这个问题是在我卸载了MySQL57后出现的，在任务管理器->服务 中找不到MySQL（MySQL5.5）。MySQL57依旧存在，可能注册表没有删除干净造成。
每次开启后问题依旧在，还需要按上述步骤开启mysqld.exe
