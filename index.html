Tomcat

查看、选择需要的Tomcat版本：https://mirrors.tuna.tsinghua.edu.cn/apache/tomcat

  • 下载： wget https://mirrors.tuna.tsinghua.edu.cn/apache/tomcat/tomcat-8/v8.5.41/bin/apache-tomcat-8.5.41.tar.gz

  • 解压：tar -zxvf apache-tomcat-8.5.46.tar.gz

查看Tomcat安装目录

  • sudo find / -name *tomcat*

创建名为“tomcat”的目录用来存放tomcat

进入目录并启动服务

安装防火墙 yum install firewalld
开启服务 systemctl start firewalld.service
启动防火墙 systemctl start firewalld
开机自启动 systemctl enable firewalld.service
查看状态 firewall-cmd –state
开启8080端口 firewall-cmd –permanent –add-port=8080/tcp
关闭防火墙systemctl stop firewalld.service
查看放行的端口 firewall-cmd –permanent –zone=public –list-ports

JDK

首先，查看是否已经预装jdk了
如果先要将其卸载，使用如下命令：
rpm -qa | grep java

查看地址

whereis java

打开配置文件

vim /etc/profile

配置内容

export JAVA_HOME=/usr/bin/java
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=".:$JAVA_HOME/lib/:$CLASSPATH"
